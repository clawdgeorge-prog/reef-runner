# Reef Runner

Reef Runner is a fresh mobile-first arcade maze-clear game built as a hard reset from the busier neon prototypes.

## Core loop
- Glide through the reef maze
- Collect every glowing pearl
- Open the ascent gate
- Reach the gate without getting tagged by jelly sentries

## Why this direction
- One very readable objective loop
- Big, high-contrast visuals that hold up on a phone
- Thumb-friendly controls: drag, swipe, or D-pad
- Simple scoring: pearls + level clear + speed bonus
- Smooth underwater motion and cohesive local art direction
- Fully self-contained: no external hosted assets

## Controls
- **Mobile:** drag anywhere on the playfield, swipe, or tap the D-pad
- **Desktop:** WASD / arrow keys
- **Pause:** P
- **Restart:** R

## Notes
- Local high score and best depth are saved in browser storage
- Audio is generated locally with WebAudio and can be toggled off
- Designed to stay playable after every iteration
